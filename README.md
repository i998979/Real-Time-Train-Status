# Real-Time-Train-Status
Real Time Train Status is an Android version of [Real Time Train Status (Web)](https://github.com/i998979/Real-Time-Train-Status-Web) on https://real-time-train-status-web.pages.dev/, that retrieves real-time train location information for East Rail Line and Tuen Ma Line, then maps it into a Google Map. Trains can be selected from the menu. Detailed information about the train will be shown by clicking the train icons.


Additionally, Next Train API and TTNT are also embedded to display the estimated arrival time and train consists (except for AEL, DRL, SIL, and TML) of the trains in the MTR network by clicking the station icons.
